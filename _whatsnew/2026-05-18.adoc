=== 簡化 Azure 中的 Cloud Volumes ONTAP 部署

現在，您可以使用快速建立功能在 Azure 中部署 Cloud Volumes ONTAP，支援單一節點和高可用性 (HA) 設定。與進階方法相比，此簡化流程減少了步驟，在單一頁面上自動設定預設值，並最大限度地減少了導覽，從而使部署更快、更輕鬆。


NOTE: 如果您使用快速建立在 Azure 中部署新的 Cloud Volumes ONTAP 系統，NetApp Console 預設會選擇 Azure 憑證。您可以使用初始 Azure 憑證部署所有 Cloud Volumes ONTAP 系統。如果您想使用其他訂閱和/或憑證，則需要切換到進階模式部署。此限制將在未來的版本中解決。
