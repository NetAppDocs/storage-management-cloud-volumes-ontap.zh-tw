Cloud Volumes ONTAP支援NetApp磁碟區加密 (NVE) 和NetApp聚合加密 (NAE)。 NVE 和 NAE 是基於軟體的解決方案，可實現符合 FIPS 140-2 標準的捲靜態資料加密。link:concept-security.html["了解有關這些加密解決方案的更多信息"] 。

NVE 和 NAE 均由外部金鑰管理器支援。
